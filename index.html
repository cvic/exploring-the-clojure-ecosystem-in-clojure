Toolbox

Clojure code quality tools

  • lein-nvd: a National Vulnerability Database dependency-checker plugin for Leiningen
  • eastwood: a lint tool that uses the tools.analyzer and tools.analyzer.jvm libraries to inspect namespaces and report possible problems
  • vizdeps: visualize Leiningen dependencies using Graphviz
  • yagni: a dead code finder (the only one of its kind, in the Clojure world)
  • lein-kibit: an analyzer which uses core.logic to search for patterns in code which can be written to be more idiomatic
  • lein-bikeshed: a Leiningen plugin designed to tell you your code is bad, and that you should feel bad
  • lucid.query: a library for querying and manipulation of Clojure source code through an xpath/css-inspired syntax
